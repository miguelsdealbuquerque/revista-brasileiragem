# 📖 Revista Brasileiragem - Edição Digital

Este repositório contém o código-fonte da versão digital e interativa da **Revista Brasileiragem**, um projeto de design editorial focado em cultura, história e identidade brasileira, desenvolvido para submissão no **Expocom**.

O objetivo deste projeto web é transpor a experiência tátil da leitura de uma revista física para o ambiente digital, garantindo alta fluidez, acessibilidade e preservação absoluta da qualidade da arte visual original.

## 🚀 Tecnologias e Soluções Adotadas

Para garantir que a revista pudesse ser lida em qualquer dispositivo sem perda de desempenho ou distorção do layout (comum em arquivos PDF pesados), o projeto foi desenvolvido como uma página web estática:

* **HTML5 & CSS3:** Estruturação semântica e estilização do layout, utilizando uma técnica de *wrapper* com `object-fit` para garantir que o rodapé e as bordas da arte nunca fossem cortados, independentemente do monitor do usuário.
* **JavaScript (Vanilla):** Lógica principal do site.
* **StPageFlip (Page Flip Library):** Biblioteca utilizada para criar o efeito físico de "virar a página" (flipbook) com aceleração de hardware e interações táteis/mouse otimizadas para 60 FPS.
